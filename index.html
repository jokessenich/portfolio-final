Our Health Web App

HTML, React, CSS, NodeJs, Express, Knex, PostgreSQL, Mocha, Chai

App Client Repo API Repo

Our Health is free web app with primarily UGC dedicated to alternative/home remedies for various health conditions. The front and backend are built from scratch using HTML 5, CSS3, React, Npm, NodeJs, Knex, PostgreSQL, Mocha, Chai.

Our Health is designed mobile-first and screen responsive. Users can easily navigate to any malady with the browse malady page without needing to login. All data is stored on a server built with Node.js on an Express framework.

Our Health allows users to easily browse conditions without logging in. However to add, remove, or "like" a remedy or malady, users must register and login.

Baby Tracks Web App

HTML, React, CSS, NodeJs, Express, Knex, PostgreSQL, Mocha, Chai

App Client Repo API Repo

Baby Tracks is a baby tracking app that allows parents to track nursing times, sleep times, and diaper frequency for newborns. The front and backend are built from scratch using HTML 5, CSS3, React, Npm, NodeJs, Express, Knex, PostgreSQL, Mocha, Chai.

Baby Tracks is designed as a mobile first and free way for mothers to track their nursing. It features a nursing timer, allowing mothers to track if the baby switches sides. It also features a "Change start time" button, if the mother missed the beginning of the nursing time.

The Dashboard allows the user to navigate to nursing timing, diaper tracking or sleep tracking. The design is simple and easy to use for the users' convenience.

Exam Calculator

HTML, JS, CSS

App

The Exam Calculator project was commissioned by the University of Michigan Math Department as a way to determine a student's projected grade. It was built with raw HTML/JS as requested by the professor to be easily copy pasted onto his editing platform. The website has been used by over 500 students

The exam calculator is a simple component, insertable into already-functioning HTML code. It takes 3 input exam score and a web homework score, and a "gateway" input and combines them into a projected score. This was my first commissioned work.

Smart Chef

HTML, JS, jQuery, CSS

App Repo

Smart Chef is my original design of an app from scratch with html, CSS, JavaScript, and jQuery. The app uses data from the 'MealDB' Api. It features responsive, semantic and accessible design.

The app has a random recipe generator and a search option. The search option generates options for recipes with their images.

The recipe page shows a high quality image of the recipe, a customizable shopping list, the recipe text and an embedded youtube video on how to make the meal.
